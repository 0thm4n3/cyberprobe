Overview

Cyberprobe contains a simple monitoring tool. It receives the ETSI protocol, analyses the protocols, and provides some simple summaries. The tool is very much a work in progress - it has very limited capability at the moment, but we've got some ideas about extending it, and making the things it does highly configurable.

Using cybermon

The cybermon program receives data from a port in ETSI LI format. It currently reports two events. Firstly, when a target is detected e.g. by a snort alert, a 'discovery' event is reported e.g. 

Target SNORTc0a80150 discovered on IP 192.168.1.80
Cybermon also tracks the amount of data travelling to destination IP addresses, and reports when considerable quantities of data start to transfer: 
Target SNORTd43af65b: IPv4 192.168.1.80 has received 0.50MB of data.
Target SNORTd43af65b: IPv4 192.168.1.80 has received 1.00MB of data.
Target SNORTd43af65b: IPv4 192.168.1.80 has received 2.00MB of data.
Target SNORTd43af65b: IPv4 192.168.1.80 has received 4.00MB of data.

Cybermon currently has full IPv4 processing, and some basic TCP and UDP processing. We're going to add more, and make the reporting configurable.
